# YouTube Downloader mit Web UI

Ein einfaches Projekt zum Download von YouTube Videos und Audio als MP4 oder MP3 mit einer Web-UI.

## Struktur

```
youtube-downloader/
│
├── backend/
│ ├── app.py
│ └── requirements.txt
│
├── frontend/
│ ├── index.html
│ ├── style.css
│ └── script.js
│
└── README.md
```

## Installation

### Backend starten

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend starten

Öffne die `frontend/index.html` Datei einfach im Browser.

## Features

- Download von YouTube Videos als MP4
- Download von Audio als MP3
- Einfache Web-UI
- CORS-Support für Frontend-Backend Kommunikation
